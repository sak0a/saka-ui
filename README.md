# Saka UI

Beautifully crafted Vue 3 components. Built with Tailwind CSS v4. Open source. Copy and paste into your apps.

Inspired by [shadcn/ui](https://ui.shadcn.com) — Saka UI gives you ownership of every component. No runtime dependency, no black boxes.

## Installation

```bash
bunx saka-ui add <component>
```

Components are copied into your project. You own the code — customize everything.

## Components

**Form Controls** — Button, Checkbox, Radio, Switch, Select, DatePicker, ColorPicker

**Data Display** — Badge, Chip, Avatar, Carousel, Accordion, Tabs, Icon

**Feedback** — Alert, Tooltip

## Development

```bash
bun install
bun run dev
```

### Build

```bash
bun run build          # Full production build
bun run build:lib      # Component library
bun run build:registry # Component registry
bun run build:cli      # CLI tool
```

## Tech Stack

Vue 3 · Tailwind CSS v4 · Vite 7 · Bun · TypeScript
